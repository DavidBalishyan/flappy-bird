# Flappy Bird Clone

A simple Flappy Bird clone built with [Love2D](https://love2d.org/) and Lua.

## Requirements

*   [Love2D 11.x](https://love2d.org/)

## How to Run

1.  Ensure Love2D is installed on your system.
2.  Navigate to the project directory in your terminal.
3.  Run the game:

    ```bash
    love .
    ```

## Controls

*   **Space** or **Enter**: Start game, Jump, Restart game
*   **Escape**: Quit the game

## Features

*   Basic physics and collision detection
*   Score tracking
*   Infinite pipe generation
